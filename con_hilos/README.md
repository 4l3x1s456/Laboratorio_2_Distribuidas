# Servidor con concurrencia – Parte 2

Este módulo implementa la versión concurrente del servidor usando hilos. También valida NRC consultando al microservicio nrcs_server.

## Características
- Concurrencia con threads
- Protección con Locks
- Validación de ID
- Validación de NRC

## Ejecución
1. Iniciar el microservicio NRC
